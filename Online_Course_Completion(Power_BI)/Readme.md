# Online Course Completion Analysis Dashboard

## Project Overview

This Power BI project analyzes online learning behavior and course completion patterns using a dataset of 100,000 students enrolled in various online courses.

The dashboard provides insights into student engagement, learning progress, course performance, and factors influencing course completion. The goal is to help educational platforms identify trends, improve student retention, and increase course completion rates.

---

## Dataset Information

**Dataset:** Course_Completion_Prediction.csv

**Records:** 100,000 Students

**Features:** 40 Columns

The dataset contains information related to:

- Student demographics
- Education level
- Employment status
- Course details
- Learning activity
- Quiz performance
- Assignment completion
- Payment information
- App usage behavior
- Satisfaction ratings
- Course completion status

---

## Key Metrics Analyzed

- Total Students
- Course Completion Rate
- Course Category Performance
- Student Engagement Metrics
- Progress Percentage
- Quiz Performance
- Assignment Submission Trends
- App Usage Analysis
- Satisfaction Ratings
- Device Usage Distribution

---

## Dashboard Insights

The dashboard helps answer questions such as:

- Which factors contribute most to course completion?
- How does student engagement affect learning outcomes?
- Which course categories have the highest completion rates?
- Does app usage impact course success?
- How do quiz scores and assignments influence completion status?
- What demographic groups show higher completion rates?
